This is a reverse engineered schematic of the Lego smart brick PCBA. All net names of the custom ASIC are guesses, I didn't do any extensive signal analysis. I also wasn't able to extract the firmware as described here: https://github.com/nathankellenicki/node-smartplay/blob/main/notes/METHODOLOGY.md, so I couldn't check how the EM9305's GPIOs are configured.

The KiCad project contains rough traces of all connections. Reference images for all 4 layers are included.
